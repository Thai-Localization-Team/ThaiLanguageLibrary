Thai Language Library is language library mod for thai language.

You can use this mod in modReferences for translate other mods.

Translation process : 75%

You can help us at  : Terraria {TH} Thai Localization (https://discord.gg/EZWusTF)

Mod Call:
- "AddTranslations"
	parameters:
		- Mod your_mod
		- List<string> files
